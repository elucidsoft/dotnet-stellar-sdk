Class FederationServer

FederationServer handles a network connection to a federation server (https://www.stellar.org/developers/learn/concepts/federation.html) instance and exposes an interface for requests to that instance. For resolving a stellar address without knowing which federation server to query use Federation#resolve(String). See Federation docs: https://www.stellar.org/developers/learn/concepts/federation.html
