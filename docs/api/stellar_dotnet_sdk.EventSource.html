Class EventSource

An EventSource represents a long-lived HTTP connection through which a Web server can “push” textual messages. To use these “Server Sent Events”, pass the server URL to the EventSource() constructor and then register a message event handler on the resulting Event Source object. The EventSource attempts to be resilient to transitory network errors and interruptions by automatically retrying connections to maintain persistence.
